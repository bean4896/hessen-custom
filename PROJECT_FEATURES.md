## üè† Project Overview

**Hessen Custom Furniture Configurator** is a modern e-commerce platform specializing in online bedframe configuration and sales. The project adopts the latest technology stack to provide a complete shopping experience, from product configuration to payment completion.

## üöÄ Core Features

### 1. Product Configurator
- **Real-time Configuration Interface**: Users can select different product options through an intuitive interface
- **Multi-dimensional Customization**: Supports configuration across materials, sizes, headboards, bases, and more
- **Instant Price Calculation**: Real-time total price calculation based on user selections
- **Configuration Saving**: Users can save multiple configuration schemes for future reference and purchase

#### Configuration Options Include:
- **Material Selection**: Rubberwood, Pinewood, Teakwood, Oakwood, Black Walnut, Cherrywood, Ashwood, Hinoki, HPL Laminate
- **Surface Finish**: Natural, Honey, Coal, White
- **Bed Sizes**: Single, Supersingle, Queen, King
- **Headboard Types**: Panel, Slat, Upholstered, None
- **Base Types**: Platform, Slats, Storage, Adjustable
- **Optional Accessories**: Premium Mattress, Bedding Set, Matching Nightstand, Extended Warranty

### 2. Product Gallery System
- **High-quality Image Display**: Supports multiple image formats
- **Configuration Preview**: Real-time product display updates based on user selections
- **Responsive Design**: Adapts to various device sizes
- **Interactive Interface**: Users can click to view products from different angles

### 3. Shopping Cart System
- **Add Products**: Add configured products to cart
- **Quantity Management**: Support for modifying product quantities
- **Price Summary**: Real-time cart total display
- **Local Storage**: Cart data saved locally, persists after page refresh

### 4. User Authentication System
- **Multiple Login Methods**: Supports email login
- **Session Management**: Secure session management
- **Permission Control**: Distinguishes between regular users and admin privileges
- **Personal Center**: User profile management

### 5. Payment System Integration
- **Stripe Integration**: Leading international payment processing platform
- **Secure Payments**: PCI DSS compliant security standards
- **Multiple Payment Methods**: Supports credit cards, debit cards, and more
- **Payment Confirmation**: Complete payment flow and confirmation mechanism

### 6. Order Management System
- **Order Creation**: Automatic order record generation
- **Order Status Tracking**: Real-time order status updates
- **Order History**: Users can view historical orders
- **Order Details**: Detailed order information display

### 7. Admin Dashboard
- **Order Management**: View and manage all orders
- **Customer Management**: Customer information management
- **Product Management**: Product configuration and pricing management
- **Data Analytics**: Sales data and statistical information
