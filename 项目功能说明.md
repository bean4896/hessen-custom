# Hessen定制家具配置器 - 项目功能说明

## 🏠 项目概述

**Hessen定制家具配置器** 是一个现代化的电商平台，专注于定制床架的在线配置和销售。该项目采用最新的技术栈，提供完整的购物体验，从产品配置到支付完成的全流程服务。

---

## 🚀 核心功能

### 1. 产品配置器 (Product Configurator)
- **实时配置界面**: 用户可以通过直观的界面选择不同的产品选项
- **多维度定制**: 支持材质、尺寸、床头板、底座等多维度配置
- **即时价格计算**: 根据用户选择实时计算总价格
- **配置保存**: 用户可以保存多个配置方案，方便后续查看和购买

#### 配置选项包括：
- **材质选择**: 橡胶木、松木、柚木、橡木、黑胡桃木、樱桃木、白蜡木、桧木、HPL层压板
- **表面处理**: 天然、蜂蜜色、炭色、白色
- **床尺寸**: 单人床、超单人床、双人床、大床
- **床头板**: 平板式、板条式、软包式、无床头板
- **底座类型**: 平台式、板条式、储物式、可调节式
- **可选配件**: 高级床垫、床上用品套装、配套床头柜、延保服务

### 2. 产品展示系统 (Product Gallery)
- **高质量图片展示**: 支持多种格式的图片展示
- **配置预览**: 根据用户选择实时更新产品展示
- **响应式设计**: 适配各种设备尺寸
- **交互式界面**: 用户可以点击查看不同角度的产品图片

### 3. 购物车系统 (Shopping Cart)
- **添加商品**: 将配置好的产品添加到购物车
- **数量管理**: 支持修改商品数量
- **价格汇总**: 实时显示购物车总价
- **本地存储**: 购物车数据保存在本地，刷新页面不丢失

### 4. 用户认证系统 (Authentication)
- **多种登录方式**: 支持邮箱登录
- **用户会话管理**: 安全的会话管理
- **权限控制**: 区分普通用户和管理员权限
- **个人中心**: 用户个人信息管理

### 5. 支付系统 (Payment Integration)
- **Stripe集成**: 国际领先的支付处理平台
- **安全支付**: 符合PCI DSS安全标准
- **多种支付方式**: 支持信用卡、借记卡等多种支付方式
- **支付确认**: 完整的支付流程和确认机制

### 6. 订单管理系统 (Order Management)
- **订单创建**: 自动生成订单记录
- **订单状态跟踪**: 实时更新订单状态
- **订单历史**: 用户可查看历史订单
- **订单详情**: 详细的订单信息展示

### 7. 管理后台 (Admin Dashboard)
- **订单管理**: 查看和管理所有订单
- **客户管理**: 客户信息管理
- **产品管理**: 产品配置和价格管理
- **数据分析**: 销售数据和统计信息

## 🛠️ 技术特性

### 前端技术栈
- **Next.js 15**: 最新的React框架，支持服务端渲染
- **TypeScript**: 类型安全的JavaScript超集
- **Tailwind CSS**: 现代化的CSS框架
- **Radix UI**: 高质量的UI组件库
- **Lucide React**: 美观的图标库

### 后端技术栈
- **Next.js API Routes**: 服务端API接口
- **Prisma ORM**: 现代化的数据库操作工具
- **PostgreSQL**: 强大的关系型数据库
- **NextAuth.js**: 完整的身份验证解决方案

### 数据库设计
- **用户管理**: 用户信息、会话管理
- **产品配置**: 产品选项、价格配置
- **订单系统**: 订单、订单项、支付信息
- **购物车**: 购物车项目和状态管理

### 部署和基础设施
- **Vercel部署**: 全球CDN加速
- **Supabase数据库**: 云端PostgreSQL服务
- **Docker支持**: 容器化部署
- **环境配置**: 开发、测试、生产环境分离
